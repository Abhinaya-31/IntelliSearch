## AI-Powered Semantic Search Engine

IntelliSearch is a smart AI-powered semantic search platform that enables users to search using natural language. By understanding context and intent rather than just keywords, it delivers precise and meaningful results.

The platform goes beyond traditional keyword matching by using vector embeddings to understand semantic meaning. When a user enters a query, it is converted into a vector and compared with stored document vectors using cosine similarity.

MongoDB Atlas is used as the backend database for storing documents and performing fast vector similarity search, while React powers the frontend interface and analytics dashboard.
